## 📌 About This Project

**HRMIS ABA Bank UI** is a **Laravel-based frontend project** that replicates the **ABA Bank login interface**.

This project focuses purely on **UI/UX design and frontend implementation**, showcasing how real-world banking interfaces can be built using Laravel Blade.

⚠️ **Note:** This is a **frontend-only project (UI only)** — no backend authentication or real banking functionality is implemented.

---

* 🎨 Practice building **real-world UI (banking interface)**
* 🧠 Improve **frontend skills using Laravel Blade**
* 🏗️ Understand structure of enterprise-style login systems

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/hrmis_ababank.git
cd hrmis_ababank
```

### 2. Install dependencies

```bash
composer install
npm install
npm run dev
```

### 3. Setup environment

```bash
cp .env.example .env
php artisan key:generate
```

---

## ▶️ Run the Project

```bash
php artisan serve
```

Open:
👉 [http://127.0.0.1:8000](http://127.0.0.1:8000)
